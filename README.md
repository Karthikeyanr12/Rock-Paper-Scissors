# ✊✋✌️ Rock Paper Scissors Game

## 📌 Description
This project is an interactive **Rock Paper Scissors game** built using **HTML, CSS, and JavaScript**. It features a clean UI with emoji-based visuals and allows users to play against the computer.

The project focuses on practicing **Advanced JavaScript Concepts, DOM manipulation, and browser storage concepts**.

---

## 🛠 Tech Stack
- HTML5  
- CSS3  
- JavaScript  

---

## ✨ Features
- 🎮 Play Rock, Paper, Scissors against the computer  
- 📊 Score tracking (Wins, Losses, Ties)  
- 💾 Persistent score using **LocalStorage** (saved even after reload/close)  
- 🔁 Reset score functionality  
- ⚡ Auto Play mode (game plays automatically)  
- 🎯 Instant result display (Win/Lose/Tie)  
- 🎨 Clean and responsive UI  

---

## 🎯 What I Learned
- DOM manipulation and event handling  
- Implementing game logic in JavaScript  
- Using **LocalStorage** to persist data  
- Handling user interactions dynamically  
- Structuring UI with HTML and styling with CSS
